# 台股漲幅排行量能篩選器

本工具會抓取 Yahoo 股市漲幅排行，分別分析：

- 上市漲幅前 100
- 上櫃漲幅前 100
- 上市 + 上櫃合併分析

篩選條件是「5 日均量大於 20 日均量，且此狀況連續 2 到 5 個交易日」。符合條件後，系統會以 MACD、KD、布林通道、量價推估籌碼與流動性做 AI 風格評分與推薦理由。

## 執行

```bash
npm start
```

開啟：

```text
http://localhost:4173
```

## 功能

- Yahoo 漲幅排行解析
- 逐檔抓取 Yahoo Finance 日 K
- 計算 5 日均量、20 日均量、MACD、KD、布林通道
- 顯示價量、MACD、KD、籌碼推估圖
- AI 推薦前 5 檔與推薦理由
- 本機快取，避免重複抓取同一批資料

## 資料限制

分點進出與大戶持股屬於交易所、集保或券商授權資料。本版先以公開日 K 價量建立「大戶買盤推估」圖，並保留 `server.js` 中的資料彙整位置，後續可接入授權 API 後替換為真實籌碼資料。

本工具僅供研究與篩選輔助，不構成投資建議。
